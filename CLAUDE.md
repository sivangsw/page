## 📋 הוראות כלליות

### פקודת "תעלה למוצר"
כאשר סיון אומרת **"תעלה למוצר"**, הכוונה היא לבצע את הפעולות הבאות:
1. `git add .` - הוסף את כל השינויים למעקב
2. `git commit -m "תיאור השינויים"` - בצע commit עם תיאור מתאים
3. `git push origin main` - העלה ל-GitHub Pages
