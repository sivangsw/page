### 1. Push to GitHub

- Source: Deploy from a branch
- Branch: main
- Folder: / (root)
- Save

### 3. Verify Deployment
- Check repository settings for Pages status
- Wait 1-2 minutes for deployment
- Visit: `https://[username].github.io/page/`
